SaltyModder è un complesso programma che permette di aggiungere dei cheats (godmode) all'interno di Brawlhalla ed una mappa custom in AmongUs (solo se pagate 10 euro potete accedervi)

![saltymodder](https://user-images.githubusercontent.com/83040047/163344474-5f7d2ab4-2869-44a7-9e84-951cec561aa3.png)

**[!]La mod di Brawlhalla influisce con le meccaniche di gioco, per cui è considerato cheating.**                                         
**[!]La mod di AmongUs va contro le policy Innersloth.**

Quando avrò 100 euro sul mio account PayPal continuerò ad implementare nuovi cheats. Il programma non è in grado di auto-aggiornarsi, quindi c'è bisogno di riacquistare il software ogni qual volta ci sarà una modifica.

Se riscontrate problemi o bug nell'utilizzo, problemi vostri. Se provate a contattarmi, la vostra email verrà cestinata e il vostro indirizzo IP tracciato.
